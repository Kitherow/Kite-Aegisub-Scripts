# Kite Aegisub Scripts

A collection of Aegisub automation scripts for timing, quality control, typesetting, clip work, gradients, image conversion, snapshots, transform effects, and color management.

## DependencyControl

DependencyControl update information is provided in `DependencyControl.json`. The feed includes the public macros listed above, the `kite.Timing` timing module, and the `kite.UI` shared settings module. Stable preferences are stored by namespace in `?user/config/kite.settings.json`.

Feed URL:

```text
https://raw.githubusercontent.com/Kitherow/Kite-Aegisub-Scripts/main/DependencyControl.json
```
