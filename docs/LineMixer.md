# Line Mixer 1.0.0

Line Mixer shuffles text among selected dialogue lines.

Menu root: `Line Mixer`

Hotkey path: `: Kite Hotkeys :/Line Mixer/Execute`

Namespace: `kite.LineMixer`

Line Mixer is a direct top-level macro with no dialog.

## Behavior

- Reads the text from selected dialogue lines.
- Shuffles the text order.
- Writes the shuffled text back into the same selected line slots.

## Requirements

Line Mixer declares no additional modules.
