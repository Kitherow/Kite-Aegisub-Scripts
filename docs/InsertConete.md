# Insert Coñete 1.0.0

Insert Coñete clones selected dialogue lines and replaces the visible text in each clone with `Coñete`.

Menu root: `Insert Coñete`

Hotkey path: `: Kite Hotkeys :/Insert Coñete/Execute`

Namespace: `kite.InsertConete`

Insert Coñete is a direct top-level macro with no dialog.

## Behavior

- Inserts one clone after each selected dialogue line.
- Preserves timing, style, actor, layer and leading override tags.
- Sets the clone text to `Coñete`.

## Requirements

Insert Coñete declares no additional modules.
