# Moka Shape 0.1.4

Moka Shape converts Adobe After Effects mask data into ASS clips or drawing lines.

Menu root: `Moka Shape`

Hotkey path: `: Kite Hotkeys :/Moka Shape/Execute`

Namespace: `kite.Mokashape`

## Output modes

- `clip`
- `iclip`
- `shape`

## Workflow

Paste AE mask data, choose placement and frame mapping, then apply the generated path to selected subtitle lines or inserted output lines.

## Configuration

Settings persist through ConfigHandler in `kite-mokashape.json`.
