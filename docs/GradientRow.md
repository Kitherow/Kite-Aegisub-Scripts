# Gradient Row 1.6.4

Gradient Row creates adaptive color gradients across selected lines and visible text.

Menu root: `Gradient Row`

Hotkey path: `: Kite Hotkeys :/Gradient Row/Execute`

Namespace: `kite.GradientRow`

## Features

- Gradient generation from palette colors or inline color states.
- Per-line and per-character gradient modes.
- Optional SubInspector integration when available.
- Saved palette colors are preserved between runs.

## Configuration

Settings persist in the Aegisub user directory through the script configuration handler.
