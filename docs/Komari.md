# Komari 1.0.0

Komari marks selected dialogue lines that contain stutter patterns.

Menu root: `Komari`

Hotkey path: `: Kite Hotkeys :/Komari/Execute`

Namespace: `kite.Komari`

Komari is a direct top-level macro with no dialog.

## Behavior

- Scans visible text in selected dialogue lines.
- Detects repeated-letter hyphen patterns such as `O-Oye`.
- Adds `Komari` to `Effect` when a stutter is found.

## Requirements

Komari declares no additional modules.
